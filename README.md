# 盒子．道南生活網路誌

這是「盒子．道南生活網路誌」的網站封存檔案。「盒子」是作者就讀國立政治大學新聞系期間，於 1997 年製作的網路多媒體電子刊物，以政大校園生活為主題，由地下兵工廠製作。刊物自民國八十六年四月創刊，至民國八十七年一月停刊，共出刊七期。

網站線上瀏覽：https://zonble.github.io/nccubox/

真的是糟糕透了。

這個什麼平裝版一片五十元、精裝版一片八十元的「國立政治大學七 十週年校慶紀念光碟」真的是糟糕透了。甚至教人不禁懷疑會不會比 從政大毀容部、或是莊大衛零號設計師手底下出來的髮型還要糟糕。

雖然說這個年頭用五十元的價格可以買到一片可能具有深刻紀念意義 的多媒體光碟－最起碼對於一個政大人來說，政大難得出了一片校慶 紀念光碟，而且政大的七十週年校慶也的確也只有可能有這麼一次。 但是說真的，我倒是反比較期待這片光碟能夠再做得仔細一些、精緻 一些，即使價錢即使會貴一些，而今天這樣的成品，我實在想要乾脆 多存幾個五十元，然後去買一片「星際大戰」的紀念光碟。

在這片校慶紀念光碟中，使用超文件方式，製作介紹政大種種的多媒 體環境，包括政大的校史、未來發展、立校精神、系所資料等等…… 總資料量佔 547Mb 大小，還算是相當紮實。

但是當我們拿到這片光碟的時候，所碰到的第一個問題就是：我們應 該要怎麼去使用？

【難道放個瀏覽器軟體這麼困難嘛】

今天若使用超文本製作文件，必然要使用超文件瀏覽軟體方能閱讀／ 瀏覽當中的文字圖形聲音資料，但是在這片光碟的使用說明當中，就 是這麼輕描淡寫一句「啟動您硬碟上的瀏覽器，在 File 選單上選擇 Open，將光碟機根目錄下的 index.htm 檔案載入即可。 」奇怪，為 什麼這片光碟的執行製作「本校網頁小組」（奇怪，怎麼從來沒有聽 過這個單位），會這麼的篤信，拿到這片光碟的政大師生、或是其他 的使用者，硬碟當中一定會有瀏覽器軟體，然後輕便巧妙的操作這樣 一個步驟？

即使在網際網路－特別是全球資訊網－被炒做得這麼熱門的今日，網 路瀏覽器軟體也不見得是使用者電腦中必然安裝的軟體。假如一名使 用者家中／宿舍中裝了一台電腦，而他並沒有數據機或網路卡等裝置 上網，平時就是拿電腦打打報告印印東西，有空的時候就玩玩接龍採 地雷，他是不會需要安裝一套網景或是微軟的瀏覽軟體的。而且在微 軟的視窗三點一以及早期的視窗九五版本中，瀏覽器也並不是作業系 統中的基本軟體。

再加上現在的瀏覽器軟體做的日益「肥大」，無論在安裝前或是安裝 後，都需要佔用相當的硬碟空間；而光碟中的第張網一頁就使用了爪 哇程式以及客戶端影像地圖的超文件語法，勢必要使用「肥大」的網 景或是微軟的瀏覽器晚近版本。想從學校將這些軟體帶回家不是一片 磁片就可以解決的事情，而想透過數據機以及電話線將這些程式下傳 回來，更是，呵，慢慢等吧！

甚至有同學放在宿舍、有上網的電腦，也都不一定有安裝瀏覽器軟體 ，比方說那位新聞系官子程同學。

既然要瀏覽商學院虛擬院館，都會副上一套 SuperScape 瀏覽軟體， 那麼光碟在製作時，為什麼也不把那些可以在網路上任意下傳的網景 、微軟瀏覽器燒錄到光碟當中？難道這樣非常困難嗎？

【資料整理大有問題】

我們費盡艱辛，終於有了軟體瀏覽光碟之後，第二個問題是：天！為 什麼資料會這樣呈現？有誰能夠明確告訴我什麼「影片館」、「圖片 館」、「多媒體館」還有「聲音館」裡頭的內容究竟為何？請不要說 「影片館理頭就是影片嘛」。

在「影片館」裡頭有一段「未來展望」的 Mpeg 影片，但是在首頁的 「未來展望」段落當中又放了一堆影片、包括校長一行人在新竹縣長 范振宗陪同下一起探勘政大第二校區的記錄等等。同樣是未來展望， 為什麼要放在同一個地方呢？

「圖片館」裡頭真的是包羅萬象，教人始終不懂的是：為什麼所有的 行政、系所單位的，圖文並茂的超文件資料會放在「圖片館」裡頭？ 「圖片館」裡頭還放有一段很好聽、似乎是重新編曲過的政大校歌， 雖然說真的很好聽，但是為什麼校歌的聲音檔會出現在「圖片館」裡 頭，那麼「聲音館」又是要拿來做什麼哩？

「多媒體館」就更是奇怪了。「多媒體館」裡頭還真的什麼都有，與 「圖片館」、「未來展望」不知道重複了多少內容。在「多媒體館」 的影像地圖中，未來展望、政大精神等等，所指定的連接位置都是相 同的，那麼又何必分成那麼多「館」。

其實，根本就不需要分成那些名稱不同實質相同的「館」，直接在首 頁分成「簡介」、「政大精神」、「人物」、「歷屆校長」、「重大 事件」、「各單位資料」、「未來展望」即可，又好記又好懂。

【一堆毛病，多得教人不想提】

又有時候，相當懷疑這張光碟的製作者到底有多用心，怎麼會作出以 下這麼多毛病出來？

首先是光碟的建議事項「螢幕設定值」建議 640X480，但是實際使用 時卻發現並非如此，許多系所的超文件資料就說要用 800X600，甚至 有個系還要求使用者使用五八六一三三、十五吋螢幕還有高級護目鏡 ，呃…我們應該要聽各系說的呢？還是要聽光碟的建議事項呢？還是 說光碟的建議事項僅適用於極少部份，換在光碟中的其他頁不一定適 用，反正都是你家的事。

又，有些系要求使用者用網景瀏覽器，又有些系要求使用微軟瀏覽器 ，為什麼瀏覽同一張光碟，我們要一直將瀏覽軟體換來換去？

光碟中的系所資料都是由各系自行製作，然後再通通使用 NCCUCD 這 個帳號，將資料複製到一台位在計算機中心、叫做 NC 的區域網路伺 服器。在尊重各系的前題下這個作法不壞，各系所也的確也不同的風 格，但是：計中是否曾經要求各系統一規格、統一在哪個螢幕環境下 瀏覽、使用哪個瀏覽軟體？還是說每個系交來之後就算了，反正通通 燒進光碟裡。

然後新聞系介紹老師的部份一堆照片沒有顯示出來；廣告系的次目錄 下有一堆無用的 UNIX 文字檔案，不知情者點下去就只看到一堆亂碼 ；然後有一大票超文件語法寫錯、超文件標示變成文字顯示出來的， 拉拉雜雜，實在是數不完。

【紀念光碟在紀念什麼？】

不過以上所說的問題似乎都沒有這個問題來的大，就是：我們到底要 拿這片七十週年校慶紀念光碟紀念什麼？這片光碟真的讓我們覺得有 什麼紀念意義嗎？

在生硬刻板的政大校史、在網路上可以隨意取得的各系所超文件介紹 、破碎片段政大校景拼湊出的影像檔案當中，我所見到的不過就是制 式的，那種提供給外賓看的那種無聊簡報，我看不到在七十週年校慶 此時此刻，我們對政大應有的感情、七十年從大時代走來的血淚、我 們在這裡的一同奮鬥、一同努力，我看不到那些我覺得值得紀念的種 種記憶。

或許吧！這片光碟製作的相當用心，甚至為了效果、還不惜斥資租用 直升機拍攝政大校景，但是我所更期待的，是另一種的用「心」：用 「心」思考對於七十週年校慶，我們所期待的是什麼；用心思考我們 的政大是什麼，還有我們在政大的嗔痴愛恨、我們的快樂與悲傷、我 們的激情與沮喪。七十週年校慶光碟，該是讓我們在若干年後，重新 將碟片放入光碟機時，我們能得以拾起記憶中的笑意，在這笑意中， 我們可能還流淌著些許的淚光…

只不過，現在的這片光碟，只讓人比失望還要來得更失望一些。

【期待的該是下一張光碟】

七十週年校慶的所有活動都已經過去了，我想校方也不可能回收這批 光碟重新製作，我們所見到的所有爛資料，也將永遠的燒錄在光碟當 中，永永遠遠。

下一次發行紀念光碟不知道會是什麼時候，也許是八十週年、九十週 年、一百週年，在那我們已經不知道政大或是我們是否存在的不可測 遙遠未來。

只是期待，下一張校慶光碟，真的有些能讓我們紀念的東西。

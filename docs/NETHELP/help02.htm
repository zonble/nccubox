本站建議的瀏覽配備

「盒子」預設的瀏覽器是微軟探險家 Internet Explorer 3.0、以及 網景 Netscape Navigator 3.0， 所以建議您使用這兩種目前在網路 上使用普及的瀏覽器軟體瀏覽「盒子」，不需要再另外掛上任何附加 項（ Plugins ），就可以達到作者預設的效果。

「盒子」中採用 JavaSript，如浮動工具列等，Netscape Navigator 2.0 和微軟 Internet Explorer 3.0 以上的版本都有支援， 但有些 視覺效果必須是 Mozilla 3.0 規格以上的瀏覽器才能看得到， Netscape 從 3.0 開始便已經支援 Mozilla 3.0，而 Internet Explorer 則在 4.0 開始支援。

因為「盒子」中使用「窗框」多重文件的瀏覽環境，您的瀏覽器最好 能支援「窗框功能」。如果您用的是 Netscape Navigator 2.0 以上 的軟體， 例如 Cello、Mosaic 等，雖然也可以看到「盒子」的內容 ，但效果並不完美。

關於播放超文件的背景音樂， Internet Explorer 在 2.0 版以上均 有支援，Netscape 3.0 以上也支援的相當完整，不過 2.x 版便支援 得並不怎麼徹底。

在 Push 技術上，雖然 Internet Explorer 及 Netscape Navigator 的 4.0 版都有支援，不過「盒子」目前僅完成 Internet Explorer 的訂閱功能，將在短期之內完成 Netscape Navigator 的訂閱功能。

「盒子」並沒有使用許多網路上常見的一些文件格式，如 VRML、 、Shockwave 以及動畫影片，所以您不需要另外安裝觀看這些檔案 的程式。不過未來將考慮加入Adobe Portable Ducument Format（PDF） 文件格式的出版品，您可能需要安裝Adobe Arcobat Reader 閱讀。

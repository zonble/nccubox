為什麼文字顯示不正常？

如果您瀏覽時發生文字顯示不正常，請參考以下說明：

1. 文字顯示是亂碼：請檢查您的作業系統是否可以處理 BIG-5 內碼 中文，如果不行的話，請更新為中文化的作業系統，或是請試著在您 的作業系統上加掛中文語文套件， 例如微軟就發行瀏覽器 Internet Explorer 3.0 的各國語文套件，以及像「雙橋」之類的軟體。 或是 請檢查您的瀏覽器解碼方式是否為 BIG-5 中文內碼。

2. 文字排列不正常： 在使用 Netscape Navigator 軟體時常會發現 文章不會自動斷行，這是因為您的瀏覽器解碼方式不正確所致。請將 內碼的解碼設定從預設的英文 ASCII 內碼改為 BIG-5 中文內碼便可 解決。 Internet Explorer 中文版由於預設值便是 BIG-5 內碼，所 以不會有這樣的問題。

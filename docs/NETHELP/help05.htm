如何訂閱盒子？

「盒子．道南生活網路誌」支援全球資訊網 Push 技術， 您只需在 http://server99.dormA.nccu.edu.tw/BOX/ABOUT/DOCS/CHANNEL/.htm 便可 以將「盒子」登記為您的頻道，加入您的頻道列，您以後觀賞「盒子 」的內容就可以像是觀賞電視頻道一樣。

「盒子」目前採用 CDF （ CHANNEL/ Definition Format ）標準，您 必須安裝微軟 Internet Explorer4.0 版或是 PointCast 的 PCN 軟 體，才能夠訂閱「盒子」的內容，目前並不支援 Netscape Navigator 4.0 的 Push 技術。如果您用的是 Netscape Navagator 或是舊版的 Internet Explorer，系統會自動不讓您使用訂閱功能。

目前坊間電腦資訊雜誌討論 Push 技術相當廣泛，您可以從中了解更 多的資訊。

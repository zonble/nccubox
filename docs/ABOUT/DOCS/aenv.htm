製作「盒子」的硬體平台

「盒子」的編輯使用一台叫做「地下兵工廠」的個人電腦編輯「盒子」，而這 台「地下兵工廠工作站」自八十四年九月開始在在「地下兵工廠實驗室」服役 。想到那時候以新台幣五萬餘元購入此機的時候，Pentium 100、16MB RAM以及 850MB的「超大硬碟」，不知道引起多少新聞系同學的側目哩，但是過了不久， 就馬上變成落伍機種。

製作「盒子」使用的軟體

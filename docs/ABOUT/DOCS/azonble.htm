楊維中

楊維中現年二十一歲，就讀於國立政治大學新聞系四年級。擅長的興趣包括刊物製作、畫圖、 電腦網路等，不擅長的興趣還有很多。楊維中的衣櫥裡有七件襯衫四條牛仔褲還有一套西裝 ，喜歡的摔角手是美國WWF聯盟的Undertaker、新日本聯盟的獸神，還有從日本轉往美國WCW 聯盟發展的蝶野正洋。

目前最大的心願是中風的母親好過來。對未來的憧憬是找個好女人嫁了。畢業後的打算是先 去中華民國陸軍窩兩年。畢生的志業是為國捐軀(如果這個國家值得這麼做的話，哈哈！！) 。

zonble

你可以將zonble翻譯成肉飽、肉包或是膿包，重點是zonble這個字典裡頭查不出意思的字， 代表的是楊維中。zonble原來是楊維中的漫畫筆名，後來亦成為他在網路上的代號。所以你 會在楊維中的塗鴉作品上看到像「199x produced by zonble」的簽名，而要用電子郵件連絡 楊維中也應該寄到zonble@jour.nccu.edu.tw這個地址。

地下兵工廠

地下兵工廠就是位在復興南路某處屬於楊維中的閨房。裡頭有張充作書桌的大辦公桌、一只 放滿不像是新聞系學生讀的書的書櫃、一張來過同學都很愛在上面躺兩下的蠕蠕的床、還有 個塞了襯衫牛仔褲及一套西裝的衣櫥。地下兵工廠裡頭有一台五八六多媒體電腦也叫做地下 兵工廠，是楊維中用來打報告玩遊戲以及製作「盒子」的地方。

【盒子支援頻道功能！】

「盒子．道南生活網路誌」支援微軟的「頻道」（CHANNEL/）功能。 透過「頻道」功能，您可以在「盒子」更新資料時，自動將資料抓回 您的電腦，同時瀏覽網頁就像是看切換頻道電視一樣。您不妨檢查 一下您的瀏覽器是不是Internet Explorer 4.0。

您必須安裝微軟探險家 Internet Explorer 4.0 以及Active CHANNEL/ 軟體才能使用「頻道」功能。目前「盒子」並不支援 Netscape NetCaster的頻道功能

【現在就加入您的頻道！】

請點選旁邊的圖示，就可以將「盒子」加入您的頻道當中！感受不同於 以往瀏覽網頁的特殊效果！如果您對「盒子」的頻道功能有任何建議， 請寄信到： zonble@jour.nccu.edu.tw

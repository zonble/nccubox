「政大需要一本雜誌！」

政大的刊物數量非常多，多到讓那幾家政 大承印廠商如奇昌、士淇打字行或是學校附近其他的打字影印店每天 的工作多的做不完。行政單位有印不完的文書資料招生簡章宣傳品、 系所單位有印不完的公告校友通訊、教授研究生有印不完的論文報告 、學生社團學會有印不完的刊物通訊。但是－政大卻沒有一本雜誌。

自從數年前「政大青年」在校園中絕跡之 後，政大校園中就再也沒有一本多元的、學生角度的「雜誌」。中文 系製作發行的「道南文學」局限於文藝創作，而各個社團學會的刊物 又多為社員之間聯繫感情，而未能兼顧整個政大校園中一萬多名的師 生讀者。

政大需要一本雜誌，所以，我想，我們應 該要有一本雜誌。

我們應該要有一本雜誌，內容是關於我們 在政大的生活，還有我們對於道南橋這頭的感情。我們應該有一本雜 誌，內容是校園時事、校園活動、消費情報、旅遊、情感、還有許多 許多，可以讓同在政大的我們，在這本雜誌前一起笑一起哭一起歡樂 一起悲傷。

所以，我們將道南橋這頭的種種，一起放 進了一只盒子裡，而，這就是我們的盒子。所以，民國八十六年四月 ，我們推出了盒子創刊號。

「盒子」使用電子網際網路發行，主要是 因為政大校園內網路使用普及，加上編輯的經濟能力目前並沒有辦法 印行平面版；當然，那天若有能力我們還是希望能發行平面版，我們 希望看見的，是每個人手中都拿著一只盒子，打開盒子可以取出刊物 圖片甚至小玩具，打開盒子可以讓盒子裡的歡樂如金粉般灑落政大的 每一個角落。

不過，「盒子」使用網路發行或是紙上發 行，都是其次的問題，最重要的是你，能不能和我們一起捧著盒子、 閱讀盒子、分享盒子。不用擔心的是，「盒子」永遠為您把蓋子開著 ，「盒子」從不上鎖，只要您願意輕輕揭開……

七月底，颱風襲擊日本。颱風在菲律賓外海形成之後，若依照原本路 徑，會直接往台灣行進，不過在脫離菲律賓海域轉為強烈颱風後，行 進路徑即不斷的向北方偏移，直襲日本。台灣還因為颱風的外圍高壓 而炎熱的晴朗了好幾天。

從日本送來的電視畫面中，可以強烈感受到颱風的威力：海浪拍打堤 岸激起的浪花高渝十尺以上、漲起的水平面將樓層淹沒、然後是多少 人失蹤多少人死亡多少人受傷，NHK 主播表情凝重。－若是這次颱風 襲擊的不是日本，而是台灣，我猜想災情可能比過去的賀伯颱風更為 嚴重。

很奇怪，這則消息在台灣的大眾傳播媒介上都沒有出現，無論是平面 媒介還是電子媒介均是如此。這則消息連彼岸的中央電視台都以相當 大的篇幅報導，可是台灣對此，連一則外電都不肯使用，連剪輯一下 日本人冒著風雨拍攝的畫面的功夫都不肯花。

我們大概也只能這麼假設：台灣人真是不到黃河心不死，非要等到哪 一次颱風讓這已經千瘡百孔的島嶼更為瘡痍不堪，才會稍稍關心颱風 動態如何；或著是，台灣的記者真的很忙，忙的沒有時間理會一個在 我們的島嶼上並不存在的颱風。

台灣的記者好像真的很忙。諸多重大刑案始終未能偵破，所以上一分 鐘要關心白冰冰跑到政大傳播學院做什麼、下一分鐘彭婉如命案又有 什麼新進展，還要忙著在台北市議會張貼海報抗議議員自肥－而後他 們又突然發現，他們需要去貼海報的地方不只是台北市議會，還有新 竹市，還有好多好多地方……。

台灣這片土地上的人們真的很忙，忙著讓白曉燕撲滅口蹄疫、忙著用 修憲為白曉燕安葬、忙著在職棒賽程中下注、忙著不關心一個島嶼上 並不存在的颱風。

既然大夥都這麼忙，那麼大夥就繼續忙大夥的吧！大概大夥也已經忙 到沒什麼時間去網路上看一份沒有什麼名氣的網路刊物－一份資訊展 或是軟體應用展都不會提到的網路刊物、一份花時間看過之後並不能 獲取什麼商機的網路刊物，一份叫做「盒子」的網路刊物。

大夥盡管忙吧！而我似乎也無須在乎些什麼，反正不是每個颱風都會 轉向的。

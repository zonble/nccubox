用你的桌面玩遊戲

您曾經想過您的桌面上，除了可以點選「我的電腦」、「網路上的芳鄰」還有「資源回收筒」之外， 進行您再電腦上的各項工作之外，還可以變成歡樂有趣的遊樂場嗎？請看以下說明，讓您和您的桌面 玩遊戲，讓您樂在其中。

1.系統需求
 您必須要有Microsoft Windows95,Microsoft WindowsNT 4.0或是IBM OS/2之類，在桌面上有固定圖示 的視窗作業系統。您在Microsoft Windows3.1或是之前的Windows版本也可以勉強使用，不過必須要透 過執行視窗應用程式並最小化之後，在桌面上出現圖示，比較麻煩。
在一些擁有X-Windows的UNIX 系統上面也應該可以，不過我們還沒有去嘗試將之後的圖形檔案轉換為 X-Windows的桌面底圖，您不妨自行嘗試看看。好像Apple Macintosh電腦也應該可以，還要再試試看 ...:)
在顯示上，您必須是640x480像素或是更高的螢幕大小，顏色最少必須有256色。

2.取得桌面遊戲圖檔
 請透過底下的HyperLink取得遊戲區桌面底圖檔案。網景Netscape Navigator和微軟Internet Explorer 的2.0版本以上均有將網路首頁上的圖形檔案轉換成桌面底圖的功能。您可以透過瀏覽器直接轉換，或是 由將檔案取回之後用其他的影像處理軟體（如Adobe Photoshop，Uload PhotoImpact等）修改轉換。
若您是透過瀏覽器轉換桌面底圖，您稍後將瀏覽器軟體關閉或是最小化之後，就可以看到桌面已經成為了 遊樂區了。
[遊戲圖檔]game.JPG/213KB(218,399位元)

3.其他準備工作
 (1)準備好您的棋子。其實，要用的棋子就是您的桌面圖示，請在桌面上放置與您的遊戲人數相同的圖示。 若是圖示不夠，可以在桌面上再加一些捷徑、開起個資料夾或是其他檔案在桌面上，增加圖示數目。
若是您的圖示太多，您不妨先將一些桌面上的文件或是捷徑移開或是刪除。若是有些系統圖示無法刪除， 請用Windows程式目錄下的regedit.exe登錄編輯程式，修改 HKEY_LOCAL_MACHINE/SOFTWARE /Microsoft/ Windows/ CurrentVersion/ explorer/ Desktop/下的機碼。

您的棋子應該會是這樣:

Player1我的電腦 Player2網路上的芳鄰 Player3資源回收筒

(2)準備您的「十八豆」。請先準備幾顆骰子，一顆兩顆三顆都可以。若是您沒有骰子，您可以透過 以下的HyperLink連往我們所提供的骰子模型，請您列印後剪下黏製。
[骰子模型]cuboid.GIF/8.80KB(9,021位元)
若您還是覺得麻煩，請找一兩枝六面的鉛筆，在每一面上寫上一到六的數字，充當骰子使用。（對！就像您 讀高中的時候猜選擇題的方法一樣）。

4.開始遊戲
 其實，看到桌面的遊樂區，您大概已經知道該怎麼玩了吧。
如果還是不懂遊戲方法的話，請透過以下的HyperLink閱讀遊戲說明書。

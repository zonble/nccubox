使用瀏覽器設定桌面底圖的方法:

首先先點選以下的圖形檔案，瀏覽圖形。若您是使用網景Netscape Navigator 2.0 或是微軟Internet Explorer 2.0以上的版本就可以將圖形直接轉為桌面。將滑鼠移動到 圖形上，按下滑鼠右鍵，就會出現一個選單；若是網景Navigator，選擇「設為視窗桌面」 （中文版）或是「Set as Netscape Wallpaper」（英文版），若是使用微軟Internet Explorer，則選擇「設為桌面底色圖案」。這樣就大功告成。

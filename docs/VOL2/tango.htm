盒子五月號紀念主題曲
 探戈小黑貓

我願意便做一隻，可愛的小小黑貓。
由早晨到了黃昏，都在你身邊繞。
當你在失意時候，我陪你高聲笑。
當你在失意時候，我陪你把淚拋。

黑貓、黑貓的探戈 探戈、探戈，
我永遠永遠是你，溫柔的小黑貓。
黑貓、黑貓的探戈 探戈、探戈，
我就是你的探戈、探戈小黑貓。

此曲取自於福爾摩沙傳播有限公司（沒聽過沒關係，因為其實本刊編輯也沒有聽過這一家 傳播公司），所推出的國語懷念老歌專輯第十集--劉家昌。作詞作曲不詳，我們只知道這首 大爛歌是由劉家昌這個很爆笑的傢伙所主唱。

礙於中華民國著作法的種種鳥鳥規定，所以無法整首歌抓下來錄成MP3格式（聽說網路上有很 多人這麼做，然後有少數被抓起來？），僅節錄短短的三十秒，無法讓我們的讀者聽完整首歌 到底有多好笑。哈哈哈哈哈哈哈。

最近發現，劉家昌這個人不但歌好笑，連他拍的電影，他的電影號稱在國外影展得獎通通都很 好笑，他最近因為「白案」在各家電試台新聞節目中曝光所說的那些鬼話也很好笑。

盒子的五月試刊第二號主題曲，就是這首很好笑的「探戈小黑貓」。 哈哈哈哈哈哈哈。

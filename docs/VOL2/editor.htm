在電腦螢幕前向學妹展示「盒子」四月份創刊號，學妹看了第一眼， 就「噗嗤」一聲笑了出來，然後說：「這根本是八卦新聞報嘛！」

盒子是八卦新聞報？也許吧，雖然我實在不了解這個把月來在校園或 是社會上流行的這個形容詞到底是什麼意思。所謂的八卦究竟是伏羲 氏在千年前畫的那些符號嗎？（我想應該不是這麼意思）有人說像獨 家報導、翡翠雜誌那種八開本雜誌又溼又鹹的標題就是八卦，有些人 告訴我閒閒沒事去渲染別人的閒事就叫做八卦，有些人晚上打電話來 找我八卦，上次我同學說有個外系的女生好八卦……

不過說來說去，我還是不知道什麼叫做「八卦」。所以，我也不太懂 「盒子」到底是哪裡八卦。

「盒子」四月份創刊號推出時正值春假期間，由於140.119.193.99 這台主機正在更換機器與配備，以及一陣子的電腦電源供應故障，在 加上並沒有用力宣傳，這些原因或許使得您無法在四月間順利的閱讀 「盒子」四月份創刊號，讓您見到該期的「盒子」是不是真的很…呃 …八卦？

不管到底八不八卦，「盒子」五月份第二期「失去一種觀視的角 度」想要表現的是政大的周遭環境，包括了猴山岳、恆光橋等等。而 同時，還是維持「盒子」四月份創刊號的文字圖形風格。

還有就是，「盒子」不會向政大政治系的首頁那樣，必須使用五八六 一三三、三十二MB記憶體、還要很好的護目鏡才能看。「盒子」絕 不標榜「新電腦萬歲」，而是為電腦設備並非頂級的使用者著想，「 盒子」不是那種追求莫名其妙的花俏效果而要求使用者把辛辛苦苦血 汗錢放進資本家口袋裡頭的那種網路首頁。

出刊前信筆所至於此，大概就是這個樣了。

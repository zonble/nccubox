政大許多科系都紛紛開設了超文件編寫課程，例如新聞系的多媒體製 作、廣告系的廣告與網際網路，資科系資管系也有相關課程。

在全球資訊網發燒的今日，這類課程除可讓學生多一種打工賺錢的管 道之外，還可以在所謂「電腦文盲」充斥的政大校園，讓學生拍拍胸 膛大聲說： 「老子（或老娘）會寫 Homepage！ 」－即使這群會寫 Homepage 的人與「電腦文盲」還是有著很大的重疊。

此類課程多循序漸進，從歐洲量子物理研究室研究全球資訊網開始， 進而講述什麼是超文本、超鏈結，之後就上機說明什麼是〈H1〉、〈 H2〉、〈P〉、〈BR〉等超文本標記，還有怎樣插入圖形、安裝CGI， 最後在期中或是期末交出一份作業，作為教師給分的依據。

一般而言，相信學生修習超文本編寫相關課程的原因，是為了抓住全 球資訊網的時代熱潮。但是，全球資訊網站的建立需要文字的編寫、 電腦影像處理、程式設計、網站規劃、網路管理、系統管理等，而課 程中對此所提卻相當有限。僅學習過超文本語法技術的學生，會寫網 頁卻不知如何將資料傳上工作站的學生，真有足夠的技術能力建立站 台，掌握時代？

將超文本本身視為一種資訊技術的教學沒什麼不好，最起碼我們的確 將很多學生的可怕自傳大頭照生活照紀念照送上網路，好讓政大校園 看來網路發展蓬勃非常。超文本的確是門技術，但卻並不只是技術。 特別在政大，電腦科系佔少數，學生對網路認識不周的環境下，更應 重視技術之外的知識、觀念的教學。

說穿了，超文本不過就是種文件格式，與微軟 Word 的文件格式 doc ，Adobe 的可攜式文件 PDF 本質上沒兩樣。 而精通 Word 裡每一項 功能的人不一定可以寫出好文章，熟悉超文本語法的人做出的網頁也 不一定好，作出優秀的超文本，有技術之外更重要的東西。

好的網頁裡最重要的就是內容，國立政治大學以人文社會科學著稱， 但是在政大的網站上似乎卻人文氣息缺乏的可憐，除了始終沒有像「 南方電子報」、「烘培姬」之類的電子刊物外，連政大師生都出走到 清大當女性主義站的站長。難道政大人除了可怕的自傳大頭照之外， 就沒有別的文字圖形可以送上網路了嘛？政大的人文精神呢？

在我們的教學當中，應該扭轉技術至上的觀念，轉向對內容的人文關 心。而教師給分依據上，也該從畫面美觀程式特效上轉為是否言之有 物，一張網頁要做的漂亮做的炫可以透過輕鬆的抄襲，但是內容的深 度與原創所需花費的功夫與帶給網路上讀者的滿足，卻遠勝前者。

而另一項重要觀念的就是：全球資訊網的建立製作，每個環節都是今 日社會嚴密分工之下的專業，網站不會是由對於網站每個部份都精通 的超人一手包辦（因為這種人是絕對的少數），而是由專業文字工作 者、插畫家、攝影師、電腦美術人員、程式設計師、網路管理人員與 系統裡人員所組成的工作團隊。學習網頁編寫最重要的，就是要認清 自己的專業何在，我們不需要求精於美術的科系學生寫 Java 程式， 也不需要求資訊科系設計畫面圖像。

各種人員如何溝通協調合作，也應是教學的重點。教授者應該教學生 認識別人的專業，並且尊重別人的專業。教授如何討論網站內容的核 心概念與呈現方式、如何共同激發創意、如何團結合作群策群力。但 是現在的教學內容，就是要學生作出一份自己的作業，交不出來就沒 有分數，從不告訴學生什麼是專業與分工。

如何找齊工作人員，塑造網站的文字圖像風格，還有網站裡的「編輯 室文化」也是嚴肅的學問，但在專業分工觀念盡無的情形下，更是遑 論以上幾項了。

我們現在僅有的，就是技術至上的教學，從某系開給外系修習課程所 設立的門檻可以明顯見到此心態。門檻是：修該課之前必須先修過計 算機概論，所持理由為編寫超文本是電腦進階「技術」，所以必須要 有基礎電腦能力。說真的，編寫超文本有多困難？有多進階？如果要 編寫超文本要有基本能力，那麼學生應該要先修的，還必須該有繪圖 、視覺傳播、美學、文學、出版概論、色彩學、心理學、符號學！

大學不該祇是職業訓練或技術養成，超文本的出現，可以帶動文字寫 作的革命與新傳播時代的無限可能，這些都不是在技術層面上所能想 像的。政大師生是否應該想想，我們可以在網路上做些什麼，我們如 何可以在網路上，展現屬於政大的一番色彩。

願你我共勉之。

〈沒有題目的造型實驗〉製作範例
本範例是在Microsoft Windows95環境下，使用Adobe Photoshop 3.05影像處理 軟體操作。硬體配備為Pentium一百，記憶體40MB。

1.先啟動Adobe Photoshop，並且開啟一個新檔案（File->New）。

如果是要作為桌面底圖之用，可以輸入六四零乘四八零、八百乘六百等等， 解析度設定在72就可以了；若是作為印刷用途則必須輸入更高的解析度，視 輸出所使用的器材而定。

2.填滿畫面底色（Edit->Fill），本範例所使用的底色為R0、G128、B128。 在「顯像紀念館」中使用的是R255、G155、B55。

然後，使用選單中的Filter->KPT 3.0->KPT Spheroild Designer 3.0... 當然，您必須先安裝KPT 3.0。

3.這是KPT Spheroild Designer 3.0...的使用操作介面，您可能需要多接 觸一下才知道每一個按鈕的用途。我們設定小球的顏色為鮮黃色，與底色成 對比色。

將小球的排列狀態設定為Radial Spherize，球的大小視需要調整。

4.從KPT Spheroild Designer 3.0...按下確定之後，就可以看到小球從畫面 中心擴散的畫面。

再按按幾下 Ctrl+F ，或是在 Filter 下作操作一下 KPT Spheroild Designer 3.0... 可以讓小球的層次更多更密，不過大概多做四五次就夠了，以免畫面雜亂。

5.製作小球的陰影，可以使用遮照或是Layer的方式來作，我們選擇透過 Alien Skin 的 Filter 來做。效果差不多，不過比較快。

首先必須做出小球的遮照。透過Select->Color Range 選擇底色後用 Select-> Invert 即可。

6.使用Filter -> Alien Skin -> Drop Shadow 之後的介面。選擇 Typical 即可 。您也可以依據您的喜好與需要自行設定陰影的各項數值。

設好的話，看見那個大大的 OK 沒有？按下去吧！

7.完成陰影之後的效果，不錯吧！其實已經接近最後的完成品了，這個時候的圖形 也可以直接拿來使用。

下一步要做的是將一個個小球變成扭曲的短線，使用的是 Filter -> Distort -> Ripple ，在畫面上面加些漣漪。

8.完成了。最後可以再修飾一下色相彩度明度上面的一些偏差。

「顯像紀念館」當中則是再將整個圖形往左邊移動了一些，並且打上了「盒子」的刊頭文字 等等。
